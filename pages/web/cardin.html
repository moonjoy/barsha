Description

This is a personal practice website project of the singer Charlotte Cardin. I used css animations and TweenMax for the smooth transitions of the blocks and the texts in the landing page.



Wireframe

The site is divided up in 3 major sections.



Layout and CSS Animation

Bootstrap framework used for the layout. Background color and images are added in their respective blocks.
Blocks sliding animated in CSS with psedo elements and keyframes.


Text Animation | Watch Demo

TweenMax used for text animation. For each text the y-axis is offset by 20 with time delay added to each.
