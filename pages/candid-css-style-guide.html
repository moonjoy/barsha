Candid CSS style guide

This global stylesheet focuses on basic styles including typography, links, buttons, colors, and basic form elements. Structure CSS is not included.



Typography

Our signature type family is Algebra. Its squarish shapes give an impression of solidity and history, while its crisp terminals keep it fresh and contemporary. Our secondary type family is used for small text such as captions and annotations. Akkurat is legible when small.

h1 is 3.75rem, 60px

h2 is 3rem, 48px

h3 is 2.25rem, 36px

h4 is 1.5rem, 24px

h5 is 1.125rem, 18px bold

p is 1.125rem regular. The line height of both h5 and p text is 1.5em.

Icons

Material Icons are the default icon font we use. See Material Icons Guide for instructions on how to integrate the icon font with your projects. See this page for a library of icons and their codes.

On this page, the icon font is imported as a Google Web Font with a line of HTML: 

				<link href="https://fonts.googleapis.com/icon?family=Material+Icons" rel="stylesheet">

To add an icon in HTML:

				mood
